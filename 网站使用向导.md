# 网页使用指南

## 安装与启动

### 环境要求

- **Python 3.8+**（推荐 3.10+）
- **Node.js 18+**（仅初次启动需要，如已预构建前端可跳过）

### 第一步：安装 Python 依赖

```powershell
pip install -r backend/requirements.txt
```

### 第二步：启动

双击 `start.bat`，脚本会自动：
1. 检测前端是否已构建（已预构建则跳过）
2. 启动后端服务（端口 8080）
3. 自动打开浏览器

如需手动启动：

```powershell
cd backend
python main.py
```

然后浏览器打开 `http://127.0.0.1:8080`

### 如果前端未预构建

需要先构建一次前端（需要 Node.js）：

```powershell
cd frontend
npm install
npx vite build
cd ..
```

之后每次启动只需运行 `start.bat` 或 `python backend/main.py`。

## 整体流程

网站分为四个步骤，顶部导航栏可切换：

```
Step 1（异物库管理）→ Step 2（数据集配置）→ Step 3（数据生成器）→ 管理（数据集管理）
```

## Step 1：异物库管理

这一步用来**提取和管理异物素材**（如包装袋中的杂质、异物颗粒等）。

### 界面布局

- **左侧**：素材资源区，包含两个标签页
- **右侧**：素材编辑区，用于抠图处理

### 操作流程

1. **上传图片**：在"待处理图像"标签页点击"上传原图"或"导入文件夹"，选择 X 光异物原图
2. **抠图提取**：点击左侧缩略图，右侧出现编辑区，可用画笔/橡皮擦/多边形工具标注异物区域
3. **保存异物**：编辑完成后点击"保存"，异物进入"异物库"
4. **管理异物**：在"异物库"标签页可修改类别、搜索、删除、导出 mask 压缩包
5. **一键清空**：每个标签页都有红色"一键清空"按钮，可批量删除

### 直接导入异物

在"异物库"标签页点击"上传单个异物"或"导入文件夹"，会弹出类别选择框，选择类别后直接导入（自动生成全图 mask）。

### 编辑区操作

| 操作 | 说明 |
|------|------|
| 画笔 | 涂抹标注异物区域（绿色半透明） |
| 橡皮擦 | 擦除已有标注 |
| 多边形 | 点击添加顶点，Enter 完成闭合，Ctrl+Z 撤销一点 |
| 滚轮 | 缩放画布 |
| 空格+拖拽 | 平移画布 |

---

## Step 2：数据集配置

这一步用来**创建和配置训练数据集**。

### 操作流程

1. 点击"+ 创建数据集"，输入名称、选择输出格式（YOLOv5/v8）
2. 新数据集自动跳转到 Step 3
3. 点击已有数据集卡片可进入 Step 3 继续生成
4. 数据集卡片支持**重命名**和**删除**（删除会同步清理关联的背景图和样本）

---

## Step 3：数据生成器

这一步用来**批量合成训练数据**。

### 界面布局

- **左侧**：素材资源区（背景图 / 异物库）
- **中间**：实时预览画布
- **右侧**：生成参数面板

### 操作流程

1. **选择数据集**：确保顶部显示当前操作的数据集名称
2. **上传背景图**：在左侧"背景图"标签页上传 X 光背景图（支持图片+同名 .txt 标注）
3. **设置生成区域**：
   - **自动检测**：选中背景后自动检测包装袋的倾斜区域（绿框）
   - **手动框选**：点击"矩形区域"拖拽，或"多边形区域"点击绘制
   - 点击"🤖 自动检测区域"可重新检测
4. **选择异物**：切换到"异物库"标签页，点击异物选中（蓝色高亮），支持全选/取消全选/拖拽到画布
5. **设置参数**：调整异物数量、缩放、旋转、边缘留白、融合模式、标注框策略等
6. **预览**：点击"🎲 随机预览"查看效果
7. **生成**：
   - **追加生成**：在已有数据基础上追加新图片
   - **覆盖生成**：清空已有数据后重新生成

### 生成参数说明

| 参数 | 说明 |
|------|------|
| 异物数量范围 | 每张图上放置的异物个数随机范围 |
| 异物缩放范围 | 异物缩放比例，50%=缩小一半，150%=放大1.5倍 |
| 异物旋转角 | 异物随机旋转的角度范围 |
| 融合模式 | direct=直接叠加；poisson=亮度匹配融合 |
| 边缘留白 | 异物与生成区域边缘的最小距离（像素） |
| 目标框策略 | tight=紧贴目标；expand=按比例向外扩张标注框 |

### 预览画布操作

| 操作 | 说明 |
|------|------|
| 滚轮 | 缩放预览 |
| 拖拽 | 平移预览 |
| 画矩形区域 | 点击"矩形区域"后在画布上拖拽 |
| 多边形区域 | 点击"多边形区域"后点击添加顶点，Enter 闭合 |
| 删除异物（手动模式） | 选中后按 Delete |
| 清空区域 | 点击"✕ 清除区域" |

---

## Step 4（管理）：数据集管理

这一步用来**浏览和导出**已生成的数据集。

### 界面布局

- **左侧**：数据集列表
- **中间**：图片预览 + 标注框显示
- **底部**：翻页、删除、导出按钮

### 操作流程

1. **预览**：左右翻页浏览生成图片，标注框以蓝色矩形叠加显示
2. **删除**：
   - 删除样本：删除当前显示的图片
   - 删除全部：清空整个数据集
3. **撤销删除**：顶部红色"↩ 撤销删除"按钮，按数据集独立恢复
4. **导出 ZIP**：点击"📦 导出 ZIP"下载完整数据集
   - `images/` 文件夹：合成图像（JPEG）
   - `labels/` 文件夹：YOLO 格式标注（txt）
   - `labels/classes.txt`：类别文件（内容: `0`）
5. **继续生成**：点击回到 Step 3 追加生成

### 小框自动扩张

面积小于生成区域 0.1% 的标注框会自动以中心等比扩张，避免训练时被忽略。

# 命令行操作指南

## 功能

批量异物生成脚本：从背景图文件夹中随机选取 X 光背景，自动检测异物生成区域，随机放置异物素材，合成图像并输出 YOLO 格式标注。

## 快速开始

```powershell
python backend/batch_generate.py --bg_dir ./backgrounds --obj_dirs ./masks_by_category/dot ./masks_by_category/ring --output ./output --total 100 --obj_count_min 2 --obj_count_max 10 --viz_boxes
```

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `--bg_dir` | str | **必填** | 背景图文件夹路径（支持 jpg/png/bmp/webp） |
| `--obj_dirs` | str[] | **必填** | 异物素材文件夹列表，可用空格分隔多个异物文件夹 |
| `--output` | str | `./output` | 输出根目录 |
| `--total` | int | 100 | 生成图片总张数 |
| `--obj_count_min` | int | 1 | 每张图最少生成异物个数 |
| `--obj_count_max` | int | 5 | 每张图最多生成异物个数 |
| `--scale_min` | int | 50 | 异物缩放最小百分比（%） |
| `--scale_max` | int | 150 | 异物缩放最大百分比（%） |
| `--rotation_min` | int | 0 | 异物旋转最小角度（°） |
| `--rotation_max` | int | 360 | 异物旋转最大角度（°） |
| `--edge_margin` | int | 40 | 异物与生成区域边缘的最小距离（px） |
| `--blend_mode` | str | `direct` | 异物融合模式：`direct`（直接融合）/ `poisson`（亮度匹配融合） |
| `--bbox_strategy` | str | `tight` | 标注框策略：`tight`（紧贴）/ `expand`（按比例扩张） |
| `--bbox_expand` | int | 10 | 标注框扩张比例（%，`expand` 策略时生效） |
| `--max_dim` | int | 400 | 自动检测区域时的降采样最大边长（越大越精细但越慢） |
| `--viz_boxes` | flag | 关闭 | 开启后输出异物自动生成区域可视化图像 |

## 输出结构

```
output/
├── images/              # 合成图像（JPEG）
│   ├── 0001.jpg
│   ├── 0002.jpg
│   └── ...
├── labels/              # YOLO 标注文件
│   ├── 0001.txt
│   ├── 0002.txt
│   ├── classes.txt      # 类别文件（内容: 0）
│   └── ...
└── backgrounds_box/     # 检测区域可视化（需 --viz_boxes）
    ├── xxx_box.jpg
    └── ...
```

## 使用示例

**基本用法：**
```powershell
python backend/batch_generate.py --bg_dir ./backgrounds --obj_dirs ./objects/条状
```

**完整参数：**
```powershell
python backend/batch_generate.py `
    --bg_dir ./backgrounds `
    --obj_dirs ./objects/点状 ./objects/条状 ./objects/块状 `
    --output ./my_output `
    --total 500 `
    --obj_count_min 2 `
    --obj_count_max 8 `
    --scale_min 30 `
    --scale_max 200 `
    --rotation_min 0 `
    --rotation_max 360 `
    --edge_margin 40 `
    --blend_mode direct `
    --bbox_strategy expand `
    --bbox_expand 15 `
    --viz_boxes
```

**代码集成：**
```python
from batch_generate import batch_generate, GenerationConfig

config = GenerationConfig(
    bg_dir="./backgrounds",
    obj_dirs=["./objects/条状", "./objects/点状"],
    output_dir="./output",
    total_count=200,
    viz_boxes=True,
)
batch_generate(config)
```
